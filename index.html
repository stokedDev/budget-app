Budget

call budgetResults(yrSalary, monthlyExpenses) in console

How To Use

  1. Open the index.html file in a web browser.
  2. Navigate to the web browser developer tools.
  3. Find the JavaScript console.
  4. In the console of the opened index.html file, call the budgetResults function by writing budgetResults(yrSalary, monthlyExpenses)
  5. Replace the function parameters yrSalary and monthlyExpenses with year income number and monthly expenses number respectively.
  6. Press the enter key on your keyboard.

After following the above steps, income-taxed budget results should print in the console.
